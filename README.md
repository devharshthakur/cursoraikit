# Cursor AI Kit

A toolkit that provides a set of commands and rules to help you build applications using [Cursor](https://cursor.com/home?from=agents).

## Installation

1. Clone this repository or copy the `.cursor/commands/` directory to a folder/filesystem
2. Copy and past the `.cursor` folder in your project.
3. Your are free to use and edit it as per your convinience

```bash
pnpm install
```

## Development

Format markdown and JSON files:

```bash
pnpm format          # Format files
pnpm format:check    # Check formatting
```
